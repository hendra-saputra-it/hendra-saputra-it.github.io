Web Inventory - Register
(PHP)

Bagian page Register ini sudah saya buat dengan validation sistem seperti mandatory tidaknya sebuah kolom input dan format-format yang valid (Email, Password Confirm). Untuk memasukan data dalam pembuatan akun ke database saya sudah bisa menerapkan sistem API, yang mana data dari form input akan di kirim ke API baru kemudian API tersebut akan mengakses fungsi yang ada di database, fungsi database tersebut pertama saya buat untuk memeriksa apakah ada data yang sama didatabasenya, jika data ada yang sama (Duplicate Key) maka proses pembuatan akun akan gagal, dan jika tidak ada data yang sama baru lah data tersebut dimasukan kedalam database. Setiap Tindakan dari fungsi tersebut akan menghasilkan keluaran berupa alert/pesan yang nantinya di kirim ke bagian front untuk di tampilkan kepada user. alert dipage Login ini juga sudah saya lengkapi dengan sweet alert.

Note : Keuntugan dari menggunakan sistem API ini adalah untuk keamanan database, karena dengan begitu database kita tidak akan terakses langsung oleh user melainkan harus melalui API yang ada diserver terlebih dahulu.
